# 🏫 초등학교 1학년 교실 적응 도우미 서비스 계획서 (가칭: "스스로 척척")

## 1. 서비스 개요 및 목적
- **목적:** 초등학교 1학년 학생들의 기본 생활 습관(루틴) 형성을 돕고 자기주도적 성취감을 부여
- **실행 환경:** 교실 전면 전자 칠판 (대형 터치 스크린 기능 활용)
- **주요 타겟:** 초등학교 1학년 학생 (메인 사용자) 및 담임 선생님 (관리자)
- **핵심 가치:** 글보다 모형과 색깔에 익숙한 1학년도 직관적으로 이해할 수 있는 쉬운 UI와 즉각적이고 긍정적인 디지털 피드백(칭찬 효과) 제공

## 2. 주요 기능 및 화면 구성

### 2.1. 메인 페이지 (학생용 전자칠판 뷰)
전자 칠판에 상시 띄워두고 아이들이 등교 직후나 쉬는 시간에 자유롭게 터치하는 화면입니다.

* **학생 명단 표시**
  * 반 전체 학생들의 이름(또는 번호+이름)이 큼직하고 귀여운 버튼 형식으로 배열
  * 완료한 학생과 아직 완료하지 않은 학생의 명찰 색상이나 테두리를 다르게 하여 시각적으로 구분
* **활동 체크 및 칭찬 프로세스**
  1. 학생이 전자 칠판에서 **자신의 이름을 터치**
  2. 팝업 창이 열리며 선생님이 지정해둔 **'오늘의 할 일' 목록** 표시 
     *(예: 알림장 제출하기, 겉옷 정리하기, 손 씻기, 우유 마시기 등)*
  3. 아이가 완수하고 **항목을 터치**하면 도장(또는 참 잘했어요 마크)이 찍히는 효과 발생
  4. 모든 항목을 완료하면 **화려한 축하 애니메이션**(폭죽, 메달 수여, 별가루 효과 등)과 사운드가 나오며 모달이 닫힘
  5. 메인 화면의 학생 명찰에 '완료 마크(예: 왕관 뱃지)' 표시

### 2.2. 설정 페이지 (선생님용 뷰)
선생님이 반 환경에 맞게 서비스를 세팅하는 화면입니다.

* **학생 명단 관리:** 학급 학생들의 이름 일괄 등록, 수정, 삭제 기능
* **활동 컨텐츠(루틴) 관리**
  * 아이들이 해야 할 기본 항목 추가 및 삭제
  * 한글에 익숙하지 않은 아이들을 위해 각 활동 항목별로 **직관적인 아이콘/그림 매칭** 기능 (예: 우유 아이콘, 가방 아이콘 등)
* **초기화 및 설정**
  * 일과가 끝나면 전체 학생의 완료 기록을 초기화하는 버튼
  * (선택) 출석부처럼 오늘의 활동 결과 목록을 파일이나 화면으로 요약해 보는 기능

## 3. UI/UX 디자인 전략 (초등학교 1학년 맞춤형)
* **화면 레이아웃 (키 높이 고려):** 1학년 학생들의 키가 작은 점을 감안하여, 직접 터치해야 하는 팝업창이나 버튼 등은 가급적 **화면의 하단 2/3 영역 내**에 집중되도록 설계합니다.
* **디자인 및 색감:** 부드럽고 따뜻한 파스텔 톤이나 밝고 경쾌한 원색을 사용하고, 모서리가 둥근 UI를 적용합니다.
* **마이크로 애니메이션:** 버튼을 누를 때 젤리처럼 튕기는 효과, 완료 시 팡파르 이펙트 등 아이들의 시선을 사로잡고 터치의 재미를 유발할 동적 요소를 적극 활용합니다.

## 4. 권장 기술 스택 프레임워크
전자칠판의 브라우저(크롬, 엣지 등)에서 풀스크린으로 띄워두는 **'웹 애플리케이션(Web App)'** 형태를 추천합니다.

* **Frontend (화면 및 인터랙션):** `React` (Next.js 또는 Vite) + `TailwindCSS`
  * 상태 관리 및 부드러운 UI 렌더링에 탁월합니다.
  * `Framer Motion` 같은 라이브러리를 추가해 다양한 칭찬 애니메이션을 쉽게 구현할 수 있습니다.
* **이미지 소스 및 API 연동 (에셋):**
  * `freepik.com` API 활용 (API Key: `FPSX0669118e4b7a42f5067cf4dad0fe138e`)

* **데이터 저장 (Database):** 
  * **핵심 DB:** `PostgreSQL` 사용 (데이터의 안정성과 향후 확장성 고려)
